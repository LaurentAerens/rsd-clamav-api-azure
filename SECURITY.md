# Security Policy

## Supported Versions

Only the latest release line is supported because important security fixes were introduced in `1.3.0` without breaking changes compared to `1.2.4`.

| Version | Supported |
| --- | --- |
| 1.3.x | :white_check_mark: |
| 1.2.4 | :x: |
| 1.2.3 | :x: |
| 1.2.2 | :x: |
| 1.2.1 | :x: |
| 1.2.0 | :x: |
| < 1.0.0 | :warning: Migrate away as soon as possible |

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub Issues in this repository.

When opening an issue, include:

- A clear description of the vulnerability.
- Steps to reproduce.
- Impact and affected version(s).
- Any suggested remediation, if available.
